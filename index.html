Gesture.js

Cross browser Pointer and Gesture event lib




Welcome to Gesture.js Page

Gesture.js uniformizes pointer and gesture inputs across mobile and desktop browsers. The code comes from ViniSketch Toolkit and was influenced by Pointer.js from borismus and his article.

The mains difference with Pointer.js are:
  • we do not change default add/removeEventListenener
  • we try to minimize the use of "createEvent" method that creates bugs with complexes situations
  • It supports IE10 (Pointer and Gesture)


The lib works on w8-IE10, Chrome, Safari, FireFox, Android stock browser, Android Chrome, Android FireFox, iOS, FireFox OS...

Listeners

vs.addPointerListener

Summary

addPointerListener() registers a single event listener on a single target. The event target may be a single element in a document, the document itself, a window. To register more than one event listener for the target, call addPointerListener() for the same target but with different event types or capture parameters.

Syntax

vs.addPointerListener (node, event name, listener[, capture]);
node

The event target.

event name

A string representing the event type to listen for.

listener

The object that receives a notification when an event of the specified type occurs. This must be an object implementing the EventListener interface, or simply a JavaScript function.

useCapture Optional

If true, useCapture indicates that the user wishes to initiate capture.

vs.removePointerListener

Summary

removeEventListener() allows the removal of event listeners from the event target.

Syntax

vs.removePointerListener (node, event name, listener[, capture]);
node

The event target.

event name

A string representing the event type to listen for.

listener

The listener parameter indicates the EventListener function to be removed.

useCapture Optional

If true, useCapture indicates that the user wishes to initiate capture.

Pointer / Gesture Events

Summary

List of events fired for one or multiple pointing device(s) (mouse, touch or pen), on down, move or up over an element.

Pointer Events

vs.POINTER_START

Event that is triggered when a contact touches the screen on element.

vs.POINTER_MOVE

Event that is triggered when a contact moves on the screen while over an element.

vs.POINTER_END

Event that is triggered when a contact is raised off of the screen over an element.

Gesture Events

vs.GESTURE_START

Triggered when the screen is touched on a location over this element.

vs.GESTURE_CHANGE

Triggered when the finger positions associated with the interaction moves on the screen. It will also be triggered during inertia processing.

vs.GESTURE_END

This event fires when all associated contacts are removed from the surface, and any inertia related movements have stopped.

Events Structure

PointerEvent

The PointerEvent class encapsulates information about a touch event.

Accessing Properties

pointerList

A collection of Pointer objects representing all pointers.

nbPointer

Number of pointers

targetPointerList

A collection of Pointer objects representing all pointers associated with this target.

changedPointerList

A collection of Pointer objects representing all pointers that changed in this event.

GestureEvent

The GestureEvent class encapsulates information about a multi-touch gesture. GestureEvent is a PointerEvent with more high level information related to the gesture.

Accessing Properties

scale

The distance between two fingers since the start of an event, as a multiplier of the initial distance.

rotation

The delta rotation since the start of an event, in degrees, where clockwise is positive and counter-clockwise is negative.

translate

The delta translation since the start of an event. This is a vector [x, y]

centroid

The centroid is the geometric center of the pointers position. Type: vs.Point object

Example

vs.addPointerListener (node, vs.POINTER_START, onPointerDown);

var listening = false;

/** Called each time a touch, mouse down, … occures */
function onPointerDown (e) {
  e.preventDefault();
  /** Set listeners for the first pointer start */
  if (!listening) {
    vs.addPointerListener (canvas, vs.POINTER_MOVE, onPointerMove);
    vs.addPointerListener (canvas, vs.POINTER_END, onPointerUp);
  }
}

function onPointerMove (e) {
  var touches = e.pointerList;
  ...
} 

/** Called each time a touch up, mouse up, … occures */
function onPointerUp (e) {
  /** Remove listeners when all pointers are up */
  if (!e.nbPointer && listening) {
    vs.removePointerListener (canvas, vs.POINTER_MOVE, onPointerMove);
    vs.removePointerListener (canvas, vs.POINTER_END, onPointerUp);
    listening = false;
  }
}
